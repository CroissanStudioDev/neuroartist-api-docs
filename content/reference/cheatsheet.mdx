# Cheatsheet

Все публичные эндпоинты gateway-а. Базовый URL: `https://api.neuroartist.ru`.

`s` — session-only · `a` — API-key или session · `—` — публичная.

## Auth (`/api/auth/*`)

| | Метод | Путь | Что |
| --- | --- | --- | --- |
| — | `POST` | `/api/auth/sign-up/email` | Регистрация |
| — | `POST` | `/api/auth/sign-in/email` | Вход |
| s | `POST` | `/api/auth/sign-out` | Выход |
| — | `POST` | `/api/auth/forgot-password/email` | Отправить ссылку восстановления |
| — | `POST` | `/api/auth/reset-password` | Установить новый пароль |
| s | `GET` | `/api/auth/get-session` | Текущая сессия |
| s | `POST` | `/api/auth/api-key/create` | Создать API-ключ |
| s | `POST` | `/api/auth/api-key/list` | Список ключей |
| s | `POST` | `/api/auth/api-key/delete` | Удалить ключ |

## Генерация — sync (`/run`)

| | Метод | Путь | Что |
| --- | --- | --- | --- |
| a | `POST` | `/run/{modelId}` | Submit + ждать результат до 5 мин |

`?passthrough=1` — оригинальные URL-ы без re-host в S3.

## Генерация — async (`/queue`)

| | Метод | Путь | Что |
| --- | --- | --- | --- |
| a | `POST` | `/queue/{modelId}` | Submit, вернёт `requestId` |
| a | `GET` | `/queue/{modelId}/requests/{requestId}` | Забрать результат |
| a | `GET` | `/queue/{modelId}/requests/{requestId}/status` | Статус (one-shot) |
| a | `GET` | `/queue/{modelId}/requests/{requestId}/status/stream` | Raw SSE статус |
| a | `GET` | `/queue/{modelId}/requests/{requestId}/progress/stream` | **Унифицированный SSE** |
| a | `PUT` | `/queue/{modelId}/requests/{requestId}/cancel` | Отмена |

`POST /queue/{modelId}?webhook=https://your.app/cb` — callback вместо polling.

## Headers

```
Authorization: Bearer na_live_xxxxxxxxxxxxxxxx
Content-Type: application/json
X-Request-ID: <uuid>                  ← в каждом ответе, прикладывайте к support
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 99
X-RateLimit-Reset: <unix-timestamp>
Retry-After: <seconds>                ← при 429/503
```

## Envelope ошибок

```json
{ "code": "insufficient_balance", "message": "Not enough credits", "details": { "required": 12, "available": 5 } }
```

Ветвите по `code`. Полный список — [Errors](/reference/errors).

## Лимиты

| Scope | Лимит |
| --- | --- |
| `POST /api/auth/sign-in/email` | 10 / 5 мин / IP |
| `POST /api/auth/sign-up/email` | 5 / 5 мин / IP |
| `POST /api/auth/forgot-password/email` | 3 / 10 мин / IP |
| `POST /webhooks/*` | 100 / 1 мин / IP |
| Генерации (`/run`, `/queue`) | 100 / час / user |
| Любая ручка через API-ключ | 1000 / час / key |
| Опционально, на ключ | `metadata.monthlyLimitRub` (₽/мес) |
| SSE-стримы | 5 одновременных / user |
