# Health и метрики

Публичные ручки для мониторинга, документации и discovery.

## `GET /`

API root metadata.

```json
{
  "name": "neuroartist-api-gateway",
  "version": "0.0.1",
  "status": "ok"
}
```

## `GET /health`

Liveness + проверка БД.

```json
{
  "status": "ok",
  "db": "up",
  "uptime": 1234567,
  "timestamp": "2026-04-25T10:00:00Z"
}
```

`status` ∈ {`ok`, `degraded`}. `db` ∈ {`up`, `down`}.

200 даже при `db: down` — gateway сам жив. Используйте поле `db` для k8s readiness вместо HTTP-кода.

## `GET /metrics`

Prometheus text format v0.0.4. Без аутентификации — защищайте network ACL на стороне оператора.

```
# HELP http_requests_total Total HTTP requests
# TYPE http_requests_total counter
http_requests_total{method="POST",path="/run/:modelId",status_class="2xx"} 142
…
```

Counters:

- `http_requests_total{method, path, status_class}` — pattern-based path
- `rate_limit_hits_total{scope}` — `ip` / `user` / `api_key`
- `billing_deduct_total{outcome}` — `success` / `insufficient` / `error`
- `sse_active_connections` — gauge

Никаких user-identifying labels.

## `GET /changelog.json`

Public changelog feed.

```json
{
  "entries": [
    { "version": "0.4.0", "date": "2026-04-25", "notes": "…" }
  ]
}
```

## `GET /openapi.json`

OpenAPI 3.1 (EN). В production-режиме отфильтрованы тэги `Admin` и `Webhooks`.

## `GET /openapi.ru.json`

OpenAPI 3.1 (RU overlay поверх EN).

## `GET /docs`

Scalar UI — interactive playground с EN/RU табами.

## `GET /llms.txt`

Token-optimized markdown для LLM-промптов.

## `GET /llms-full.txt`

Полная markdown-документация.

## `GET /.well-known/agent-configuration`

Agent-auth discovery (alias `/api/auth/agent-configuration`).

```json
{
  "version": "1.0",
  "auth": { "type": "api_key", "header": "Authorization", "scheme": "Bearer" },
  "endpoints": { "models": "/models", "run": "/run/{modelId}", "queue": "/queue/{modelId}" }
}
```
