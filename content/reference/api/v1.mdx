# `/v1`

OpenAI-shaped слой для клиентов, которые уже используют OpenAI SDK. Он не заменяет нативные `/run` и `/queue`: это короткий путь для синхронной генерации изображений.

## `GET /v1/models`

Публичный список моделей в формате OpenAI.

```bash
curl https://api.neuroartist.ru/v1/models
```

```json
{
  "object": "list",
  "data": [
    {
      "id": "fal-ai/nano-banana-pro",
      "object": "model",
      "owned_by": "neuroartist",
      "x_category": "image",
      "x_display_name": "Nano Banana Pro"
    }
  ]
}
```

## `POST /v1/images/generations`

**Аутентификация:** API-key или session.  
**Тело запроса:** OpenAI-compatible image payload.

```bash
curl -X POST https://api.neuroartist.ru/v1/images/generations \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "fal-ai/nano-banana-pro",
    "prompt": "editorial portrait, monochrome, soft daylight",
    "size": "1024x1024",
    "response_format": "url"
  }'
```

```json
{
  "created": 1714050000,
  "data": [
    { "url": "https://s3.cloud.ru/neuroartist/.../result.webp" }
  ]
}
```

`size` маппится в aspect ratio (`1024x1024` → `1:1`, `1792x1024` → `16:9`, `1024x1792` → `9:16`). `response_format: "b64_json"` скачивает результат и возвращает base64.

## `POST /v1/chat/completions`

Возвращает `501 not_implemented`: сервис специализируется на image/video/audio generation и не проксирует chat completions.
