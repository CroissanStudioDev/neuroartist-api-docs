# `/billing`

Пополнение баланса и работа с платежами. Платежи — через YooKassa.

## `POST /billing/topup`

Создать платёж на пополнение.

**Доступно только из дашборда.** Через API-ключ создать платёж нельзя.

**Тело запроса:**

```json
{
  "amountRub": 1000,
  "returnUrl": "https://gateway.neuroartist.ru/billing"
}
```

| Поле | Тип | Описание |
| --- | --- | --- |
| `amountRub` | int | Сумма в рублях. Минимум — `MIN_TOPUP_RUB` (default 500). |
| `returnUrl` | string | Куда YooKassa редиректит после оплаты. |

**Ответ 201:**

```json
{
  "payment": {
    "id": "pay_abc",
    "yookassaId": "2c…",
    "status": "pending",
    "creditsToGrant": 1000,
    "amountKopecks": 100000,
    "currency": "RUB",
    "description": "Пополнение баланса Neuroartist",
    "createdAt": "…"
  },
  "confirmationUrl": "https://yoomoney.ru/checkout/payments/v2/contract?…"
}
```

Перенаправьте пользователя на `confirmationUrl`. Зачисление происходит автоматически по webhook от YooKassa в 1–5 секунд после оплаты.

**Ошибки:**

- `400 minimum_topup` — `amountRub < MIN_TOPUP_RUB`.
- `403 forbidden` — пришли с API-ключом.
- `502 payment_provider` / `payment_provider_error` — YooKassa временно недоступна.

## `GET /billing/payments`

Список платежей.

**Параметры query:** `limit`, `offset`.

```json
{
  "items": [
    {
      "id": "pay_abc",
      "yookassaId": "2c…",
      "status": "succeeded",
      "creditsToGrant": 1000,
      "amountKopecks": 100000,
      "currency": "RUB",
      "description": "…",
      "createdAt": "…"
    }
  ],
  "total": 12,
  "limit": 50,
  "offset": 0
}
```

`status` ∈ {`pending`, `waiting_for_capture`, `succeeded`, `canceled`}.

## `GET /billing/payments/{id}`

Детали платежа.

```json
{
  "payment": {
    "id": "pay_abc",
    "yookassaId": "2c…",
    "status": "succeeded",
    "creditsToGrant": 1000,
    "amountKopecks": 100000,
    "currency": "RUB",
    "description": "…",
    "createdAt": "…",
    "succeededAt": "…",
    "paymentMethod": { "id": "pm_abc", "mask": "•••• 4242", "type": "bank_card" },
    "receiptUrl": "https://yookassa.ru/receipts/…"
  }
}
```

## `GET /billing/payments/{id}/receipt`

**Доступно только из дашборда.** 302-редирект на чек YooKassa. Чеки 54-ФЗ — в дашборде → **Биллинг → История**.
