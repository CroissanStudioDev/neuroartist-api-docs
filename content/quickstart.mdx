# Быстрый старт

<Steps>

### Создайте аккаунт

В дашборде [`gateway.neuroartist.ru`](https://gateway.neuroartist.ru) или через API:

```bash
curl -X POST https://api.neuroartist.ru/api/auth/sign-up/email \
  -H "Content-Type: application/json" \
  -d '{"email":"you@example.com","password":"MyStrongPassword2026!","name":"You"}'
```

После регистрации начисляется welcome-бонус 50 ₽ (≈ 6 image-генераций).

### Получите API-ключ

В дашборде: **Настройки → API-ключи → Новый ключ**. Ключ показывается **один раз** — сохраните.

```
na_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

Передаётся в `Authorization: Bearer …` или `x-api-key: …`.

### Найдите модель

```bash
curl 'https://api.neuroartist.ru/models?category=image&sort=popular&limit=5'
```

Каждая модель — `modelId` (содержит слеши, не URL-кодируйте), `creditCost`, JSON Schema input-а.

### Сгенерируйте

```bash
curl -X POST https://api.neuroartist.ru/run/flux/dev \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "editorial portrait, monochrome, soft daylight",
    "image_size": "portrait_4_3",
    "num_inference_steps": 28
  }'
```

```json
{
  "images": [{ "url": "https://s3.cloud.ru/neuroartist/.../result.webp", "width": 768, "height": 1024 }],
  "seed": 12345,
  "timings": { "inference": 4.21 }
}
```

Готово. URL живёт в нашем S3 пока вы его не удалите.

</Steps>

## Если не сработало

| Статус | Код | Что делать |
| --- | --- | --- |
| 401 | `unauthorized` | Ключ просрочен или невалиден — создайте новый |
| 402 | `insufficient_balance` | Пополните баланс через [`/billing/topup`](/api/billing) |
| 404 | `unknown_model` | Опечатка в `modelId` — `GET /models` для списка |
| 429 | `rate_limited` | См. `Retry-After` в headers |
| 503 | `provider_error` | Upstream флапает — повторите через `Retry-After` сек |

Полный справочник — [коды ошибок](/errors).
