# Быстрый старт

<Steps>

### Создайте аккаунт

Зарегистрируйтесь в дашборде [`gateway.neuroartist.ru`](https://gateway.neuroartist.ru). После подтверждения email на баланс начисляется приветственный бонус **50 ₽** — этого хватит на несколько image-генераций.

### Получите API-ключ

В дашборде: **Настройки → API-ключи → Новый ключ**.

```
na_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

Ключ показывается **один раз** — сохраните в env вашего приложения. Передаётся в каждом запросе:

```
Authorization: Bearer na_live_xxxxxxxxxxxxxxxx
```

### Сгенерируйте изображение

```bash
curl -X POST https://api.neuroartist.ru/run/flux/dev \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "editorial portrait, monochrome, soft daylight",
    "image_size": "portrait_4_3"
  }'
```

```json
{
  "images": [
    { "url": "https://s3.cloud.ru/.../result.webp", "width": 768, "height": 1024 }
  ],
  "seed": 12345
}
```

URL живёт в нашем S3 — забирайте напрямую, ссылка не протухает.

</Steps>

## Если не сработало

| Статус | Код | Что делать |
| --- | --- | --- |
| 401 | `unauthorized` | Ключ просрочен или невалиден — создайте новый |
| 402 | `insufficient_balance` | Пополните баланс в дашборде |
| 404 | `unknown_model` | Опечатка в `modelId` — `GET /models` для списка |
| 429 | `rate_limited` | См. `Retry-After` в заголовках ответа |
| 503 | `provider_error` | Временная проблема, повторите через `Retry-After` секунд |

Полный справочник — [коды ошибок](/errors).
