# Загрузка входных файлов

Многие модели принимают `image_url` / `video_url` / `audio_url`. Чтобы передать свой файл — сначала загрузите его в наш сервис.

```
POST /me/uploads
Content-Type: multipart/form-data
field: file
```

**Session-only.** API-ключ не принимается.

## Готовый рецепт

```bash
curl -X POST https://api.neuroartist.ru/me/uploads \
  -b cookies.txt \
  -F "file=@/path/to/photo.jpg"
```

```json
{
  "assetId": "asset_abc",
  "url": "https://s3.cloud.ru/neuroartist/uploads/u_…/asset_abc.jpg",
  "contentType": "image/jpeg",
  "sizeBytes": 487123
}
```

Используйте `url` в input модели:

```bash
curl -X POST https://api.neuroartist.ru/run/fal-ai/flux/dev/image-to-image \
  -H "Authorization: Bearer na_live_xxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "make it watercolor",
    "image_url": "https://s3.cloud.ru/neuroartist/uploads/u_…/asset_abc.jpg",
    "strength": 0.65
  }'
```

## Из браузера

```js
async function upload(file) {
  const form = new FormData()
  form.append('file', file)
  const resp = await fetch('https://api.neuroartist.ru/me/uploads', {
    method: 'POST',
    credentials: 'include',
    body: form
  })
  if (!resp.ok) {
    const err = await resp.json()
    throw new Error(`${resp.status} ${err.code}`)
  }
  return resp.json()
}

const { url } = await upload(document.querySelector('input[type=file]').files[0])
```

## Лимиты

| Параметр | Значение |
| --- | --- |
| Max size | `MAX_UPLOAD_BYTES` (10 MiB по умолчанию) |
| Image | `image/jpeg` `image/png` `image/webp` `image/heic` `image/heif` |
| Video | `video/mp4` `video/quicktime` `video/webm` |
| Audio | `audio/mpeg` `audio/wav` `audio/webm` `audio/mp4` `audio/ogg` `audio/flac` |

Превышение → `413 asset_too_large`. Неподдерживаемый MIME → `415 unsupported_mime`.

## Альтернатива — внешний URL

Если у вас уже есть публичный URL с CORS — можно сразу подсунуть в input модели **без** загрузки в наш сервис:

```json
{ "image_url": "https://your-cdn.example.com/photo.jpg", "prompt": "…" }
```

Риск на вас — если URL отвалится во время генерации, упадёт `provider_error`.

## Когда нужен upload, а когда нет

| Schema поле | Действие |
| --- | --- |
| `image_url`, `video_url`, `audio_url`, `*_url` | Нужно URL — загрузите либо подсуньте свой |
| `prompt`, `negative_prompt` (string) | Текст — никаких uploads |
| `image` (base64) — редкий случай | base64 inline в JSON |
