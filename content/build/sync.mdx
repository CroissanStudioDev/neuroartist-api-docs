# Синхронный запрос — POST /run

Sync-режим — один HTTP-запрос, один ответ. Подходит для image-генераций ≤ 30 сек.

## Готовый рецепт

```bash
curl -X POST https://api.neuroartist.ru/run/fal-ai/flux/dev \
  -H "Authorization: Bearer na_live_xxxx" \
  -H "Content-Type: application/json" \
  --max-time 320 \
  -d '{
    "prompt": "editorial portrait, monochrome",
    "image_size": "portrait_4_3",
    "num_inference_steps": 28
  }'
```

```json
{
  "images": [{ "url": "https://s3.cloud.ru/neuroartist/.../result.webp", "width": 768, "height": 1024 }],
  "seed": 12345,
  "timings": { "inference": 4.21 }
}
```

## JavaScript

```js
const NA_API_KEY = process.env.NA_API_KEY

async function generate(modelId, input) {
  const resp = await fetch(`https://api.neuroartist.ru/run/${modelId}`, {
    method: 'POST',
    headers: { Authorization: `Bearer ${NA_API_KEY}`, 'Content-Type': 'application/json' },
    body: JSON.stringify(input),
    signal: AbortSignal.timeout(320_000)
  })
  if (!resp.ok) {
    const err = await resp.json().catch(() => ({}))
    throw Object.assign(new Error(err.message ?? resp.statusText), { code: err.code, status: resp.status })
  }
  return resp.json()
}

const result = await generate('fal-ai/flux/dev', {
  prompt: 'editorial portrait, monochrome',
  image_size: 'portrait_4_3'
})
console.log(result.images[0].url)
```

## Python

```python
import os, requests

NA_API_KEY = os.environ["NA_API_KEY"]

def generate(model_id: str, payload: dict) -> dict:
    resp = requests.post(
        f"https://api.neuroartist.ru/run/{model_id}",
        headers={"Authorization": f"Bearer {NA_API_KEY}"},
        json=payload,
        timeout=320,
    )
    resp.raise_for_status()
    return resp.json()

result = generate("fal-ai/flux/dev", {
    "prompt": "editorial portrait, monochrome",
    "image_size": "portrait_4_3",
})
print(result["images"][0]["url"])
```

## Параметры

`modelId` со слешами — передавайте как есть, `fal-ai/flux/dev`. URL-кодировать не нужно.

Тело — input модели по её JSON Schema. Получить:

```bash
curl https://api.neuroartist.ru/models/fal-ai/flux/dev | jq .schema
```

`?passthrough=1` — оригинальные URL провайдера без re-host (для отладки; протухают через ~24 часа).

## Что если 5 минут не хватит

Возвращается `202 Accepted`:

```json
{ "code": "accepted", "requestId": "req_abc", "modelId": "…" }
```

Дальше — async path:

```bash
# Polling
curl https://api.neuroartist.ru/queue/fal-ai/flux/dev/requests/req_abc/status \
  -H "Authorization: Bearer na_live_xxxx"

# Или SSE
curl https://api.neuroartist.ru/queue/fal-ai/flux/dev/requests/req_abc/progress/stream \
  -H "Authorization: Bearer na_live_xxxx"
```

→ [Асинхронная очередь и SSE](/build/async).

## Обработка ошибок

```js
try {
  const result = await generate('fal-ai/flux/dev', input)
} catch (e) {
  switch (e.code) {
    case 'insufficient_balance': return showTopupModal({ required: e.details?.required })
    case 'rate_limited':         return retryAfter(e.retryAfter)
    case 'invalid_body':         return showValidationErrors(e.details?.issues)
    case 'provider_error':       return retryAfter(e.retryAfter ?? 30)
    default: throw e
  }
}
```

Полный список — [Errors](/reference/errors).
