# Биллинг и кредиты

Внутри gateway — **кредиты** (целые числа). 1 кредит ≈ 1 ₽.

## Жизненный цикл денег

```
top-up   →  balance  →  deduct (submit)  →  refund (если упало)
                              ↓
                        reconcile (по фактическому usage)
```

- **Списание** атомарно при `POST /run` или `POST /queue/{modelId}`. Не хватает — `402 insufficient_balance`, **запрос провайдеру не уходит**.
- **Возврат** — автоматически при upstream-ошибке.
- **Reconciliation** — корректировка по факту (для per-second/per-megapixel моделей).

Все движения — в `GET /me/transactions` с типами `topup` / `deduct` / `refund` / `grant`.

## Проверить баланс

```bash
curl https://api.neuroartist.ru/me/balance -H "Authorization: Bearer na_live_xxxx"
# {"userId":"u_…","balance":248}
```

## Estimate перед запуском

```bash
curl -X POST https://api.neuroartist.ru/models/fal-ai/flux/dev/estimate \
  -H "Authorization: Bearer na_live_xxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"…","num_inference_steps":28}'
# {"modelId":"…","estimatedCostRub":8,"priceUnit":"image","quantity":1,"currency":"RUB"}
```

## Top-up

**Только session.** Минимум `MIN_TOPUP_RUB` (default 500 ₽).

```bash
curl -X POST https://api.neuroartist.ru/billing/topup \
  -H "Content-Type: application/json" -b cookies.txt \
  -d '{"amountRub":1000,"returnUrl":"https://gateway.neuroartist.ru/billing"}'
```

```json
{
  "payment": { "id": "pay_abc", "status": "pending", "creditsToGrant": 1000, "amountKopecks": 100000 },
  "confirmationUrl": "https://yoomoney.ru/checkout/payments/v2/contract?…"
}
```

Перенаправьте на `confirmationUrl`. Зачисление — автоматом по webhook от YooKassa за 1–5 секунд.

## Сохранённые карты

После успешной оплаты YooKassa возвращает `payment_method_id`. Gateway сохраняет (с маской PAN) для:

- Auto-topup без переввода реквизитов.
- "Оплатить сохранённой картой" в UI.

```bash
curl https://api.neuroartist.ru/me/payment-methods -b cookies.txt
curl -X POST https://api.neuroartist.ru/me/payment-methods/pm_abc/default -b cookies.txt
curl -X DELETE https://api.neuroartist.ru/me/payment-methods/pm_abc -b cookies.txt
```

## Auto-topup

Списание с default-карты при просадке ниже порога. Подробнее — [/guides/auto-topup](/guides/auto-topup).

```bash
curl -X PATCH https://api.neuroartist.ru/me/auto-topup \
  -H "Content-Type: application/json" -b cookies.txt \
  -d '{"enabled":true,"thresholdRub":200,"amountRub":1000}'
```

## Чеки (54-ФЗ)

```bash
curl -L https://api.neuroartist.ru/billing/payments/pay_abc/receipt -b cookies.txt
# 302 → https://yookassa.ru/receipts/…
```

## Welcome-бонус и low-balance email

- При sign-up — `WELCOME_BONUS_RUB` (default 50 ₽).
- При просадке ниже `LOW_BALANCE_THRESHOLD_RUB` (default 100) — **один** email до следующего top-up.
- Отписаться: `PATCH /me {"lowBalanceEmailOptOut":true}`.
