# Генерация видео

Видео-модели — от 30 секунд до нескольких минут. Используйте **async + SSE** или **webhooks**, sync-режим (`/run`) для них почти всегда отвалится по 5-минутному таймауту.

## Минимальный пример

```js
import { fetchEventSource } from '@microsoft/fetch-event-source'

const NA_API_KEY = process.env.NA_API_KEY
const BASE = 'https://api.neuroartist.ru'

// 1. Отправить задачу
const submit = await fetch(`${BASE}/queue/fal-ai/wan-25-preview/text-to-video`, {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${NA_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: 'A cinematic shot of a snowy forest at dawn, slow camera pan',
    duration: 5,
    aspect_ratio: '16:9'
  })
}).then(r => r.json())
// { requestId: "req_abc", status: "IN_QUEUE" }

// 2. Подписка на прогресс
await new Promise((resolve, reject) => {
  fetchEventSource(
    `${BASE}/queue/fal-ai/wan-25-preview/text-to-video/requests/${submit.requestId}/progress/stream`,
    {
      headers: { Authorization: `Bearer ${NA_API_KEY}` },
      onmessage(msg) {
        const e = JSON.parse(msg.data)
        if (e.stage === 'completed') resolve(e.result)
        if (e.stage === 'failed') reject(new Error(e.error?.message))
      }
    }
  )
})
```

Результат:

```json
{
  "video": {
    "url": "https://s3.cloud.ru/neuroartist/.../result.mp4",
    "content_type": "video/mp4",
    "duration": 5.0
  },
  "seed": 42
}
```

## Популярные модели

| modelId | Назначение | Цена |
| --- | --- | --- |
| `fal-ai/wan-25-preview/text-to-video` | text-to-video, до 8 сек | 60 ₽ / 5 сек |
| `fal-ai/kling-video/v2/standard/text-to-video` | text-to-video Kling | 80 ₽ / 5 сек |

| `fal-ai/minimax/video-01-live` | MiniMax video-01 | 70 ₽ / 5 сек |

Полный каталог — `GET /models?category=video` или [/generate/models](/generate/models).

## Image-to-video

Передаёте `image_url` — сервис сделает анимацию из этой картинки.

```bash
# 1. Загрузить кадр
curl -X POST https://api.neuroartist.ru/me/uploads \
  -b cookies.txt \
  -F "file=@/path/to/keyframe.jpg"

# 2. Анимировать
curl -X POST https://api.neuroartist.ru/queue/fal-ai/kling-video/v2/standard/image-to-video \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "image_url": "https://s3.cloud.ru/neuroartist/uploads/…",
    "prompt": "slow camera dolly forward",
    "duration": 5
  }'
```

## Webhook вместо SSE (для production)

Видео-генерация может занять 1–5 минут. Если ваш runtime serverless или клиент закрывает вкладку — используйте webhook:

```bash
curl -X POST 'https://api.neuroartist.ru/queue/fal-ai/kling-video/v2/standard/text-to-video?webhook=https://your.app/cb' \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"…","duration":5}'
```

Готовый код приёмника с подписью — [приём webhook](/build/webhooks).

## Что НЕ делать

- ❌ Не используйте `POST /run` для видео — sync-таймаут 5 минут, видео часто длиннее.
- ❌ Не открывайте 10 SSE-стримов сразу: лимит 5 на пользователя. Для batch — webhook.
- ❌ Не игнорируйте `failed` стадию — провайдер может вернуть ошибку через 30 секунд после submit (например, NSFW-фильтр).

## Стоимость

Цена считается за длительность (`duration`). До запуска можно посчитать:

```bash
curl -X POST https://api.neuroartist.ru/models/fal-ai/kling-video/v2/standard/text-to-video/estimate \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"…","duration":5}'
```

```json
{ "estimatedCostRub": 80, "priceUnit": "second", "quantity": 5 }
```
