# `/models`

Каталог моделей с фильтрами, поиском и оценкой стоимости.

## `GET /models`

Список моделей с пагинацией.

**Параметры query:**

| Параметр | Тип | Описание |
| --- | --- | --- |
| `category` | string | Фильтр по категории (`image`, `video`, `audio`, …) |
| `lab` | string | Производитель (`fal-ai`, `flux`, …) |
| `tag` | string | Тэг |
| `search` | string | Подстрока в `displayName` / `modelId` |
| `sort` | enum | `popular`, `newest`, `price_asc`, `by_category` |
| `includeCommunity` | bool | Включать community-модели (default `false`) |
| `limit` | int | Размер страницы (default 50) |
| `offset` | int | Сдвиг (default 0) |

**Ответ 200:**

```json
{
  "items": [
    {
      "modelId": "fal-ai/flux/dev",
      "providerId": "fal",
      "displayName": "FLUX.1 [dev]",
      "category": "image",
      "lab": "flux",
      "tags": ["editorial", "fast"],
      "creditCost": 8,
      "discountPercent": 0,
      "priceUnit": "image",
      "thumbnailUrl": "https://…",
      "summary": "Fast, photorealistic image generation.",
      "summaryRu": "Быстрая фотореалистичная генерация."
    }
  ],
  "total": 142,
  "limit": 50,
  "offset": 0
}
```

**Аутентификация:** не требуется.

## `GET /models/{modelId}`

Детали одной модели — schema, цена, описание.

**Параметр path:** `modelId` может содержать слеши (`fal-ai/flux/dev`). Передавайте как есть.

**Ответ 200:**

```json
{
  "modelId": "fal-ai/flux/dev",
  "providerId": "fal",
  "displayName": "FLUX.1 [dev]",
  "category": "image",
  "creditCost": 8,
  "priceUnit": "image",
  "discountPercent": 0,
  "tags": ["editorial"],
  "schema": {
    "type": "object",
    "properties": {
      "prompt": { "type": "string", "description": "Text prompt" },
      "image_size": { "type": "string", "enum": ["square_hd", "portrait_4_3", …] },
      "num_inference_steps": { "type": "integer", "minimum": 1, "maximum": 50, "default": 28 }
    },
    "required": ["prompt"]
  },
  "schemaRu": { /* RU-overlay */ },
  "examples": [ { "input": { "prompt": "…" }, "output": { "images": [{"url":"…"}] } } ],
  "documentation": "https://fal.ai/models/fal-ai/flux/dev"
}
```

`schema` — JSON Schema input-а модели. Используйте для генерации форм или валидации.

**Ошибки:** `404 unknown_model`.

## `POST /models/{modelId}/estimate`

Расчёт стоимости без списания.

**Аутентификация:** API-key или session.

**Тело запроса:** input модели (тот же, что и для `/run`).

**Ответ 200:**

```json
{
  "modelId": "fal-ai/flux/dev",
  "estimatedCostRub": 8,
  "priceUnit": "image",
  "quantity": 1,
  "currency": "RUB"
}
```

`estimatedCostRub` — верхняя граница списания. Для моделей с динамической ценой (per-second video) формула учитывает параметры из тела запроса.

**Ошибки:**

- `400 invalid_body` — input не прошёл schema-валидацию.
- `401 unauthorized`.
- `404 unknown_model`.
