# Биллинг

Внутри API — **кредиты** (целые числа). 1 кредит ≈ 1 ₽.

## Жизненный цикл денег

```
top-up   →  balance  →  deduct (submit)  →  refund (если упало)
                              ↓
                        reconcile (по фактическому usage)
```

- **Списание** атомарно при `POST /run` или `POST /queue/{modelId}`. Не хватает — `402 insufficient_balance`, **запрос провайдеру не уходит**.
- **Возврат** — автоматически при upstream-ошибке.
- **Reconciliation** — корректировка по факту (для per-second/per-megapixel моделей).

Все движения — в `GET /me/transactions` с типами `topup` / `deduct` / `refund` / `grant`.

## Проверить баланс

```bash
curl https://api.neuroartist.ru/me/balance \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx"
# {"userId":"u_…","balance":248}
```

## Оценка стоимости перед запуском

```bash
curl -X POST https://api.neuroartist.ru/models/fal-ai/flux/dev/estimate \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"…","num_inference_steps":28}'
# {"estimatedCostRub":8,"priceUnit":"image","quantity":1,"currency":"RUB"}
```

## Пополнение баланса

Только для сессии (через дашборд). Минимум `MIN_TOPUP_RUB` (по умолчанию 500 ₽).

```bash
curl -X POST https://api.neuroartist.ru/billing/topup \
  -H "Content-Type: application/json" -b cookies.txt \
  -d '{"amountRub":1000,"returnUrl":"https://gateway.neuroartist.ru/billing"}'
```

```json
{
  "payment": { "id": "pay_abc", "status": "pending", "creditsToGrant": 1000, "amountKopecks": 100000 },
  "confirmationUrl": "https://yoomoney.ru/checkout/payments/v2/contract?…"
}
```

Перенаправьте пользователя на `confirmationUrl`. Зачисление — автоматически по webhook от YooKassa за 1–5 секунд.

## Сохранённые карты

После успешной оплаты YooKassa возвращает `payment_method_id`. Сервис сохраняет (с маской PAN), чтобы не вводить реквизиты повторно.

```bash
# список
curl https://api.neuroartist.ru/me/payment-methods -b cookies.txt

# сделать default
curl -X POST https://api.neuroartist.ru/me/payment-methods/pm_abc/default \
  -b cookies.txt

# удалить
curl -X DELETE https://api.neuroartist.ru/me/payment-methods/pm_abc \
  -b cookies.txt
```

## Авто-пополнение

Чтобы не получать `402 insufficient_balance` посреди пайплайна:

```
balance < thresholdRub  →  charge amountRub с default-карты  →  webhook  →  +amountRub
```

Срабатывает в момент попытки списания. Не чаще раза в 60 секунд.

<Steps>

### Сохраните карту

Заплатите один раз через `POST /billing/topup` — YooKassa вернёт `payment_method_id`, сервис его сохранит автоматически.

### Сделайте карту default

```bash
curl -X POST https://api.neuroartist.ru/me/payment-methods/pm_abc/default \
  -b cookies.txt
```

### Включите авто-пополнение

```bash
curl -X PATCH https://api.neuroartist.ru/me/auto-topup \
  -H "Content-Type: application/json" -b cookies.txt \
  -d '{"enabled":true,"thresholdRub":200,"amountRub":1000}'
```

| Поле | Диапазон |
| --- | --- |
| `thresholdRub` | ≥ `AUTO_TOPUP_MIN_THRESHOLD_RUB` (default 10) |
| `amountRub` | `AUTO_TOPUP_MIN_AMOUNT_RUB`–`AUTO_TOPUP_MAX_AMOUNT_RUB` (default 50–10 000) |

</Steps>

<Callout type="info">
  Авто-пополнение работает только из сессии — API-ключ не настроит. Это намеренно: утёкший ключ не должен включать списания с карты.
</Callout>

Выключить:

```bash
curl -X PATCH https://api.neuroartist.ru/me/auto-topup \
  -H "Content-Type: application/json" -b cookies.txt \
  -d '{"enabled":false}'
```

Если списание не прошло (карта отклонена, нет default-метода) — обычная ошибка `402 insufficient_balance` на текущем submit. Email + запись `topup_failed` в `/me/activity`.

## Welcome-бонус и уведомление о низком балансе

- При регистрации — `WELCOME_BONUS_RUB` (по умолчанию 50 ₽).
- При просадке ниже `LOW_BALANCE_THRESHOLD_RUB` (по умолчанию 100) — **один** email до следующего пополнения.
- Отписаться: `PATCH /me {"lowBalanceEmailOptOut":true}`.
