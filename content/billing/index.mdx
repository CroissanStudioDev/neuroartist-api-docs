# Биллинг

Внутри API — **кредиты** (целые числа). 1 кредит ≈ 1 ₽.

## Жизненный цикл денег

```
top-up   →  balance  →  deduct (submit)  →  refund (если упало)
                              ↓
                        reconcile (по фактическому usage)
```

- **Списание** атомарно при `POST /run` или `POST /queue/{modelId}`. Не хватает — `402 insufficient_balance`, **запрос провайдеру не уходит**.
- **Возврат** — автоматически при upstream-ошибке.
- **Reconciliation** — корректировка по факту (для per-second / per-megapixel моделей).

Все движения видны в `GET /me/transactions` с типами `topup` / `deduct` / `refund` / `grant`.

## Проверить баланс

```bash
curl https://api.neuroartist.ru/me/balance \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx"
# {"userId":"u_…","balance":248}
```

## Оценка стоимости перед запуском

```bash
curl -X POST https://api.neuroartist.ru/models/flux/dev/estimate \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"…","num_inference_steps":28}'
# {"estimatedCostRub":8,"priceUnit":"image","quantity":1,"currency":"RUB"}
```

## Пополнение

Делается в дашборде [`gateway.neuroartist.ru`](https://gateway.neuroartist.ru) → **Биллинг → Пополнить**. Минимальная сумма — 500 ₽. Оплата через YooKassa, чек 54-ФЗ формируется автоматически.

После успешной оплаты карта сохраняется (с маской PAN) — чтобы не вводить реквизиты повторно и для авто-пополнения.

## Авто-пополнение

Чтобы не получить `402 insufficient_balance` посреди пайплайна:

```
balance < threshold  →  charge amount с default-карты  →  webhook  →  +amount
```

Срабатывает в момент попытки списания. Не чаще раза в 60 секунд.

В дашборде → **Биллинг → Авто-пополнение**:

- **Threshold** — порог срабатывания (минимум 10 ₽).
- **Amount** — сумма пополнения (50–10 000 ₽).
- **Default-карта** — обязательно, выбирается отдельно.

Если списание не прошло (карта отклонена, нет default-метода) — обычная ошибка `402 insufficient_balance` на текущем submit + email-уведомление.

## Чеки и история платежей

Список платежей через API:

```bash
curl https://api.neuroartist.ru/billing/payments \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx"
```

Сами чеки 54-ФЗ и PDF-выписки — в дашборде → **Биллинг → История**.

## Welcome-бонус и уведомление о низком балансе

- При регистрации — приветственный бонус, хватит на первые тесты.
- При просадке ниже 100 ₽ — **один** email до следующего пополнения. Отписаться можно в дашборде.
