# Neuroartist API Gateway

**Neuroartist API Gateway** — публичный developer-API над генеративными моделями (image / video / audio / multimodal). Один HTTPS-эндпоинт, кредитный биллинг, прозрачный SSE-прогресс генераций и хранение результатов в нашем S3.

Под капотом — fal.ai и другие провайдеры, но вашему коду этого знать не нужно: каталог моделей, формат запросов и формат ответа стабильны.

## Что вы получаете

- **Единая аутентификация.** Cookie-сессия для дашборда, API-ключи (`na_live_…`) для серверного кода. Один токен — оба пути.
- **Каталог моделей.** Текущие версии fal.ai с RU-описаниями и стабильной ценой в кредитах.
- **Sync и async режимы.** `POST /run/{modelId}` — синхронно, `POST /queue/{modelId}` + SSE — асинхронно с прогрессом.
- **Биллинг в кредитах.** 1 кредит ≈ 1 ₽. Top-up через YooKassa, авто-пополнение по порогу, чеки.
- **Re-hosting артефактов.** Все картинки/видео переезжают из CDN провайдера в наш S3 — ссылки не протухают.
- **Webhooks.** Регистрируете URL → получаете подписанные (Svix-style) callbacks вместо опроса.
- **OpenAPI 3.1 + Scalar UI.** Машинно читаемая спецификация (RU и EN) на `/openapi.json`, `/docs`, `/llms.txt`.

## Базовый URL

| Окружение | URL |
| --- | --- |
| Production | `https://api.neuroartist.ru` |
| Staging | `https://staging.neuroartist.ru` |
| Local dev | `http://localhost:8000` |

Все примеры далее используют production-URL. Замените на ваш при локальной разработке.

## Поддержка

- Issue в репозитории gateway.
- Email команды (см. дашборд).
- Логи запросов и активности — `/me/activity` (UI и API).
