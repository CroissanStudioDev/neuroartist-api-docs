# Neuroartist API Gateway

Один HTTPS-эндпоинт к генеративным моделям (image / video / audio). API-ключ, кредитный биллинг, SSE-прогресс, S3-хранение результатов.

## За 30 секунд

```bash
# 1. Получите ключ в дашборде https://gateway.neuroartist.ru → Settings → API Keys

# 2. Сгенерируйте картинку
curl -X POST https://api.neuroartist.ru/run/fal-ai/flux/dev \
  -H "Authorization: Bearer na_live_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"editorial portrait, monochrome","image_size":"portrait_4_3"}'
```

```json
{ "images": [{ "url": "https://s3.cloud.ru/neuroartist/.../result.webp", "width": 768, "height": 1024 }], "seed": 12345 }
```

Это всё, что нужно для production использования. Ниже — задачи посложнее.
